프로젝트 필요 기능

사용자 기능
- 회원 가입(이메일 기반) 및 탈퇴
- 로그인/로그아웃/자동 로그인 설정
- 마이페이지(회원정보 조회/수정)
- 메일함(메일 번역 + 송/수신, 삭제)
- 크롤링(텍스트 기반의 요청을 처리해 관련된 텍스트, 이미지, 음성 등의 파일을 검색하고 다운로드 가능, 저장 X) 
- 문의 및 신고

친구 기능
- 사용자 검색(사용자 코드, 이메일, 필요에 따라 닉네임을 통한 검색도 가능)
- 친구 추가 및 삭제
- 친구 메모

채팅 기능
- 채팅방 생성/채팅방 정보 확인 및 수정/채팅방 삭제
- 사용자 검색 및 초대
- 채팅방 퇴장
- 채팅방 알림 설정
- 메세지 송, 수신(복수의 파일 첨부 가능)

워크샵(디스코드의 개별 서버, 카카오톡 단톡방 개념) 기능
- 워크샵 생성/정보 확인 및 수정/삭제
- 사용자 검색 및 초대, 추방
- 워크샵 퇴장
- 워크샵에서 진행할 각종 작업을 등록/확인/수정/삭제
- 워크샵 알림 설정(데드라인, 공지, 멘션 알림 등)
- 워크샵 대시보드(전체 작업 현황 시각화)

스레드(디스코드 개별 채널 + 카페 게시판 개념) 기능
- 스레드 생성/정보 확인 및 수정/삭제 
- 스레드 알림 설정(포스트, 코멘트에 대한 알림 등)

포스트(스레드에 작성된 게시물) 기능
- 포스트 작성, 수정, 조회, 삭제
- 복수의 파일 첨부

코멘트(포스트에 작성된 스레드 참가자들의 반응) 기능
- 코멘트 작성, 수정, 조회, 삭제
- 단일 파일 첨부


필수 기능

▶ LLM을 활용한 감성 분석 및 요약 기능이 지원된 하이워크 게시판 시스템
  - 스레드/포스트 내의 대화 내용을 분석 및 반응을 요약

▶ 번역 기능을 지원하는 웹메일 호스팅 시스템
  - 외부 메일과 연동, 번역 기능을 지원하는 메일 시스템

▶ 이미지 생성 서비스 기반의 메시지(SNS) 호스팅 시스템
  - 텍스트를 기반으로 한 요청을 통해 각종 파일을 생성하고 다운로드(생성 결과는 저장됨)하거나 채팅 메시지, 포스트, 코멘트에 첨부 가능

▶ Llamaindex를 활용한 고객 클라우드 서비스 지원 토크 시스템
  - 워크샵 또는 스레드에 첨부된 파일(포스트, 코멘트 포함) LlamaIndex를 활용해 검색하여 제공
